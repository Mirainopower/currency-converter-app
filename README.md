# Currency Converter App

This is a simple Python currency converter app. The app allows users to enter an amount and convert it between different currencies.

## Features

- Converts currency values
- Simple Python program
- Easy to use

## How to Use

1. Download the Python file.
2. Open the file in a Python editor such as PyCharm, VS Code, or IDLE.
3. Run the program.
4. Enter the amount and currency information.
